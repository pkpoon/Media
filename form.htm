HTML Form

Web forms allow your users to type information into form fields on a web page. For example Userid and password in login Page, Text in Blog, Name, sex or age in User Information Page.

Common form elements

All form elements must be placed inside the form tag

• Text Field: A single-line field for typing in text
• Hidden Field: A field that sends a value without your user seeing it
• Text Area: A multi-line text field for larger amounts of space
• Checkbox: Used when giving the user a choice of items (when they can choose more than one item)
• Radio Button: Used when giving the user a choice of items (when they can choose only one item)
• List/Menu
• Button
